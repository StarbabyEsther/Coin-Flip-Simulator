# Coin Flip Simulator 🪙

A coin flip simulator built in Java.

## About
The program flips a coin 50 times and counts 
how many heads and tails you get. 
Results are different every time you run it!

## What I learned
- How for loops work:
  - int i = 0 means start at 0
  - i < 50 means keep going while i is less than 50
  - i++ means add 1 each time
- Random doesn't take System.in like Scanner does,
  it just needs new Random()
- How to use counters with ++
